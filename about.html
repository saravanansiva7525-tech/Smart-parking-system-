About the Project

SMART PARKING SYSTEM WITH SLOT DETECTION & MOBILE APP is designed to reduce parking search time through real-time sensing, reservation, guidance, and reporting.

Project Overview

The system combines ESP32 controllers, IR sensors, Firebase-ready data handling, and a responsive web interface to show parking slot status and support reservations.

Objective

To provide a reliable, user-friendly platform where drivers can find available slots, reserve parking, receive alerts, and navigate efficiently to the parking zone.

Features

  • Live slot detection
  • Reservation ID generation
  • Google Maps routing
  • Notification updates
  • Parking history search

Future Scope

  • Payment gateway integration
  • License plate recognition
  • Predictive occupancy analytics
  • Native Android and iOS apps
  • Multi-location admin control
